Welcome to the official repository for the oneAPI-GenAI-Hackathon-2023 organized by Hack2Skill!

## Getting Started

To get started with the oneAPI-GenAI-Hackathon-2023 repository, follow these steps:

### Submission Instruction:
  1. Fork this repository
  2. Create a folder with your Team Name
  3. Upload all the code and necessary files in the created folder
  4. Upload a **README.md** file in your folder with the below mentioned informations.
  5. Generate a Pull Request with your Team Name. (Example: submission-XYZ_team)

### README.md must consist of the following information:

#### Team Name - Bravo
#### Problem Statement - GenAI-Powered Customer Support Optimization
#### Team Leader Email - harshsarnaik@gmail.com

### A Brief of the Prototype: We will use AI for customer service/support as a fast and efficient way .
  This section must include UML Diagrams and prototype description
  
### Tech Stack: 
   List Down all technologies used to Build the prototype
   
### Step-by-Step Code Execution Instructions:
  This Section must contain a set of instructions required to clone and run the prototype so that it can be tested and deeply analyzed
  
### Future Scope:
   Write about the scalability and futuristic aspects of the prototype developed
